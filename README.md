# Explain This Later — Drupal7 Based Personal Productivity Tool

## Overview
**Explain This Later** is a personal productivity web application that helps users revisit concepts or links they don’t fully understand.  
It is a **time-based learning reminder system**.

### Key Features
1. **User Accounts**
   - Registration, login, and role-based permissions

2. **Deferred Item Management**
   - Create, edit, delete items
   - Mark status: Pending, Learned, Dropped

3. **Reminder System**
   - Set reminder dates
   - Automated email reminders
   - Status tracking

4. **Personal Dashboard**
   - View upcoming, overdue, and completed items
   - Access full item history

5. **Calendar View**
   - View upcoming pending, completed and missed items in calendar view

6. **Progress Visualization**
   - Visualize learning progress with charts and graphs
